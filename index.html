AstroStellar (N-Body Simulation)

This project involved creating a high-performance simulation of N-body gravitational interactions, primarily aimed at modeling galaxy collisions. The core of the simulation was built in C++ for maximum performance. I implemented and tested several numerical integration schemes, including the energy-conserving Leapfrog method and the higher-accuracy Runge-Kutta (RK4) method, to solve the Hamiltonian dynamics of the system.

A key achievement was the parallelization of the force calculation algorithm, which is the primary computational bottleneck ($O(N^2)$ complexity). By leveraging multi-threading, the simulation could scale to handle over one million particles, allowing for detailed and visually compelling astrophysical scenarios. The project was presented at the CFI Open House, showcasing a simulated collision between two galaxy-like structures.

Technologies: C++, OpenMP, Gnuplot, Python (for visualization), Git

Speaker and LED Composite System

As a final project for my Analog Systems course, I designed a complete system to create a sound-reactive light show. The system takes an audio signal as input and processes it to drive both a speaker and a synchronized array of LEDs.

The design involved several stages: a DC-DC buck converter for efficient power management, active band-pass filters to separate the audio into different frequency channels (bass, mid, treble), peak detectors to measure the amplitude of each channel, and a high-efficiency Class-D audio amplifier to drive the main speaker. The entire circuit was designed, laid out, and rigorously tested within the LTSpice simulation environment, demonstrating robust and accurate performance, which earned the highest grade in the class.

Technologies: LTSpice, Circuit Design, PCB Layout (KiCAD), Class-D Amplifiers, Active Filters
